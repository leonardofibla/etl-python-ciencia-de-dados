# etl-python-ciencia-de-dados
Projeto prático de ETL com Python aplicando os conceitos de Extração, Transformação e Carregamento de dados. Utiliza leitura de CSV ou lista local, processamento com Pandas e geração de mensagens personalizadas. Demonstra organização de pipeline de dados e boas práticas para construção de soluções em Ciência de Dados.
